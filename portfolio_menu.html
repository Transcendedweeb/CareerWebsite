Nolai Dashboard

Part of the NOLAI initiative: a dashboard that combines a Unity-based VR app with a Laravel web platform. It allows researchers to guide, record, and analyze children’s language learning sessions.

Dobbelspel Unity

The upgraded version of my interactive family board game, rebuilt in Unity for richer visuals and more advanced gameplay. Players roll dice on a physical gameboard controlled by an Arduino board, move across the board, and dive into minigames like a souls-like boss fight, football penalties, and quizzes on a PC connected to the gameboard. This version expands on my original Python version to create an even more immersive New Year’s tradition.

Het Water Komt - VR

Developed in collaboration with NTR, this VR prototype brings the "Het Water Komt" podwalk into a fully virtual environment. The experience allows users to explore the story from home, making it accessible regardless of weather or mobility.

Dobbelspel Python

The first version of my interactive family board game, combining a physical gameboard with Python software. Dice rolls and movement happen on the gameboard, while minigames such as rock-paper-scissors, an osu-inspired challenge, and a quiz run on the PC. This version laid the foundation for the later Unity upgrade.

Limescope Brittenburg

An immersive mobile application build into a telescope that visualizes the long-lost Roman fortress Brittenburg. Developed in collaboration with CORPUS and Fat100%, it allows visitors to step back in time and explore history firsthand. It's placed 1 to 1 to correctly display where the fort was located in Katwijk, Netherlands.

REKTon Bossfight

My first Unity-project, a FPS level featuring a two-phase boss fight. Manage mobs, ammo, and timing in a Dark Souls-inspired challenge.

Project Violet

A personal desktop assistant and hardware monitor displayed on a small LCD. It visualizes CPU, RAM, and disk usage with dynamic graphics and interactive speedometers, while also responding to voice commands for quick system actions. Built with Python and Pygame, it combines real-time data visualization, custom animations, and voice-controlled automation in a compact, visually engaging interface.
